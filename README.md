
# Programmation Mobile - FST Tanger

Ce dépôt contient l'ensemble de mes travaux pratiques réalisés en **Licence d'Ingénierie du Développement des Applications Informatiques (IDAI)** à la FSTT.

## 📁 Liste des Travaux Pratiques

### TP 1 : Manipulation des API avec Fetch
Ce premier TP se compose de trois exercices fondamentaux pour maîtriser les requêtes HTTP :
1. **To-Do List** : Utilisation des méthodes GET, POST et PUT avec *JSONPlaceholder*.
2. **Recherche d'images de chiens** : Filtrage dynamique et affichage de données complexes depuis *Dog CEO API*.
3. **Convertisseur de devises** : Calcul de taux de change en temps réel via *Frankfurter API*.

### TP 2 : Maîtrise des Composants de Base et Mise en Page
Ce second projet porte sur la création d'une interface mobile de voyage interactive et optimisée.

#### Objectifs Réalisés :
* **Affichage Dynamique** : Utilisation du composant `FlatList` pour gérer une liste de destinations avec une performance fluide.
* **Architecture d'Écran** : Intégration d'une bannière et d'une introduction via `ListHeaderComponent` pour un défilement unifié.
* **Design Flexbox** : Alignement horizontal du titre et du badge "Populaire" grâce à `flexDirection: 'row'` et `justifyContent: 'space-between'`.
* **Stylisation Avancée** : Mise en place de cartes avec ombres portées (`elevation`), bordures arrondies (`18px`) et marges internes (`padding: 22`) pour une interface moderne.
